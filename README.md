# Developer Portfolio

Welcome to my personal developer portfolio website! This is a vibrant, interactive, and fully responsive single-page portfolio built completely from scratch using **Vanilla HTML, CSS, and JavaScript**. 

There are no heavy frameworks or build tools required—just a single `index.html` file that powers everything.

## 🌟 Features

- **Vibrant Dark Theme:** A stunning dark mode aesthetic with electric gradients and glassmorphism elements.
- **Antigravity Animations:** Smooth, scroll-triggered reveal animations and interactive floating cards.
- **Data-Driven Architecture:** All portfolio content (skills, projects, experience) is securely rendered from a central JavaScript data object, making updates incredibly simple.
- **Hidden Admin Panel:** Includes a fully functional browser-based admin panel! Press `Ctrl + Shift + A` (Password: `admin123`) to visually edit profile details and access an advanced JSON editor to update projects and skills in real-time. Changes are saved locally.
- **Fully Responsive:** Beautifully optimized for both desktop and mobile devices.

## 🛠️ Technology Stack

- **HTML5** for semantic structure.
- **CSS3** (Pure CSS) with custom styling, variables, flexbox, grid, and keyframe animations.
- **JavaScript (ES6+)** for dynamic data injection, scroll event handling, and logic.
- **Lucide Icons** via CDN for lightweight, crisp vector icons.
- **Google Fonts** ('Playfair Display' for headings, 'DM Sans' for body text).

## 🚀 How to Use / Deploy

Because this project relies entirely on client-side code, deploying it is entirely effortless:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mohanramasamy2005-eng/New-Portfolio.git
   ```
2. **Local Preview:** Simply open the `index.html` file in any modern web browser.
3. **Deployment:** You can drop the `index.html` file into any static hosting service like **GitHub Pages**, **Netlify**, or **Vercel** and your site will be live instantly. 

## ✏️ Formatting & Editing Content

To customize or update the website with your own information:

**Method 1: Source Code**
Open `index.html` in an editor. Locate the `defaultData` JavaScript object near the bottom (around line `811`). You can safely edit the text strings for your bio, skills, and projects here. Save the file and refresh!

**Method 2: UI Admin Panel**
1. Open the website in your browser.
2. Press `Ctrl + Shift + A` on your keyboard.
3. Enter the password: `admin123`.
4. Edit your content on the fly and click **Save to LocalStorage**.
